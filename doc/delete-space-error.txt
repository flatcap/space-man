*delete-space-error.txt*  Fix space errors in a file

Author:  Richard Russon (flatcap) <rich@flatcap.org>

This plugin is only available if 'compatible' is not set.
It is part of the |space-man| plugin.

INTRODUCTION                                              *delete-space-error*

'delete-space-error' will fix any space characters.
That is space characters immediately before tab characters.

MAPPINGS                                         *delete-space-error-mappings*

'delete-space-error' creates a set of <Plug> mappings.
The local mappings work on a Line, Motion or Visual region
The file mappings work on the entire File, all Args, Buffers or Windows.

Plugin mappings:

  Line    <plug>SpaceErrorL
  Motion  <plug>SpaceErrorM
  Visual  <plug>SpaceErrorV
  Args    <plug>SpaceErrorA
  Buffers <plug>SpaceErrorB
  File    <plug>SpaceErrorF
  Window  <plug>SpaceErrorW

Default user mappings:

  Line    <leader><space>ee
  Motion  <leader><space>e
  Visual  <leader><space>e
  Args    <leader>a<space>e
  Buffers <leader>b<space>e
  File    <leader>f<space>e
  Window  <leader>w<space>e

EXAMPLES                                         *delete-space-error-examples*

These examples assume the <Leader> is set to its default value of
\ (backslash) and the plugin is using its default user mappings.

  Type this... | Work on | To remove space errors from
  ------------ | ------- | -----------------------------
  \<space>l9j  | Motion  | 10 lines down
  \<space>lG   | Motion  | cursor to end of file
  \<space>lgg  | Motion  | cursor to top of file
  \<space>lip  | Motion  | this paragraph

  V9j\<space>l | Visual  | 10 lines down
  \<space>ll   | Line    | This line only

  \a<space>l   | Args    | All lines in all arguments
  \b<space>l   | Buffers | All lines in all buffers
  \f<space>l   | File    | All lines in this file
  \w<space>l   | windows | All lines in all windows

CONFIGURATION                                      *delete-space-error-config*

'delete-space-error' is controlled by two global variables which both default to 1 (true).
The first to enable the function, the second to create default user mappings.
>
  let g:space_man_enable_space_error          = 1
  let g:space_man_create_mappings_space_error = 1
<
